About

Page to Template is a little utility to convert a bulletin package to a media package containing templates created from the template-based bulletins in the package. By default the template will be named the same as the bulletin's description, but you can edit it during the conversion (you'll be prompted for each bulletin).

Usage

Simply drag a bulletin package file onto the Page To Template window. It will read the package, prompt you for the template names, then create a media package in the same directory with the word Bulletins changed to Templates in the filename for each bulletin package dragged onto the window.
